* Purpose
This document is the *single source of truth* for homelab services, users, ports, and IDs.

Goals:
 * Avoid port collisions
 * Reuse consistent user/group IDs
 * Make adding new services trivial and predictable
 * Serve as design + operational documentation

This file should be consulted *before* adding any new service, container, or systemd unit.
