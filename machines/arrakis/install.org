#+TITLE: Installing NixOS on arrakis
#+AUTHOR: Masood Ahmed
#+OPTIONS: toc:nil num:nil

This document describes the end-to-end process for installing NixOS on the
=arrakis= laptop using:

- NixOS Live ISO (booted via USB)
- LUKS-encrypted ext4 with LVM (swap + nix)
- tmpfs root with impermanence (ephemeral root, persistent /home)
- systemd-boot + systemd initrd + Plymouth boot splash
- Flake-based configuration

* Overview

The installation flow:
- Boot the NixOS Live ISO via USB
- Connect to WiFi
- Partition and format the disk using =disko= (LUKS + LVM + ext4)
- Generate SSH + age keys
- Install NixOS using the flake
- Reboot into the installed system

* Prerequisites

- NixOS Live ISO on USB (built via =just build-iso=)
- WiFi credentials for initial setup
- GitHub repo with NixOS flake (=github:masood09/nix=)

* Boot NixOS Live ISO

Write the ISO to USB and boot the laptop from it. The live ISO already
contains your SSH public key as an authorized key.

* Connect to WiFi

The live ISO uses NetworkManager. Connect using =nmcli=:

#+begin_src bash
nmcli device wifi list
nmcli device wifi connect "<SSID>" password "<PASSWORD>"
#+end_src

Verify connectivity:

#+begin_src bash
ping -c 3 github.com
#+end_src

* SSH into the Live Environment (Optional)

From another machine on the same network:

#+begin_src bash
ssh nixos@<LAPTOP_IP>
#+end_src

Switch to root:

#+begin_src bash
sudo su - root
#+end_src

* Identify the Disk

Find the disk ID for the laptop's NVMe:

#+begin_src bash
ls -la /dev/disk/by-id/ | grep nvme
#+end_src

Note the disk ID (e.g., =nvme-Patriot_M.2_P300_512GB_...=). This must match
the ID configured in =homelab.disks.root= in =machines/arrakis/default.nix=.

* Partition and Format Disk with Disko

Run Disko using the flake configuration for =arrakis=:

#+begin_src bash
nix --experimental-features "nix-command flakes" \
  --option tarball-ttl 0 \
  run --refresh github:nix-community/disko -- \
  --mode destroy,format,mount \
  --flake "github:masood09/nix#arrakis"
#+end_src

During LUKS container creation, you will be prompted for the encryption
passphrase. Choose a strong passphrase -- this will be required on every
boot to unlock the disk.

Disko creates the following layout:
- EFI partition (1 GiB, vfat) at =/mnt/boot=
- LUKS container (=cryptroot=) with LVM inside:
  - =vg-swap= (40 GiB) -- swap for hibernate
  - =vg-nix= (remaining space) -- ext4 at =/mnt/nix=
- tmpfs at =/mnt= (root, 20 GiB)

Verify:

#+begin_src bash
mount | grep mnt
lsblk
#+end_src

* Prepare Persistent and Secret Directories

Create the directory structure required by impermanence and secret handling:

#+begin_src bash
mkdir -pv /mnt/nix/{secret/{initrd,age},persist/{etc/ssh,home}}
chmod 0700 /mnt/nix/secret
#+end_src

* Generate SSH and age Keys

Generate the SSH host key used in initrd:

#+begin_src bash
ssh-keygen -t ed25519 -N "" -C "" \
  -f /mnt/nix/secret/initrd/ssh_host_ed25519_key
#+end_src

Generate an SSH keypair used for age encryption:

#+begin_src bash
ssh-keygen -t ed25519 -N "" -C "" \
  -f /mnt/nix/secret/age/ssh_ed25519_key
#+end_src

Convert the SSH public key to age format:

#+begin_src bash
nix-shell --extra-experimental-features flakes \
  -p ssh-to-age \
  --run 'cat /mnt/nix/secret/age/ssh_ed25519_key.pub | ssh-to-age'
#+end_src

Save the resulting age public key -- it must be added to =.sops.yaml= in
the flake repository and secrets re-encrypted before the first deploy.

* Generate hardware-configuration.nix (First Install Only)

Only needed if the hardware has changed or this is a brand-new machine.
arrakis already has a checked-in =hardware-configuration.nix=. Skip this
step if reinstalling on the same hardware.

#+begin_src bash
nixos-generate-config --root /mnt --no-filesystems --show-hardware-config \
  > /tmp/hardware-configuration.nix
#+end_src

Compare with =machines/arrakis/hardware-configuration.nix= in the flake
repository. If hardware has changed, update and commit before proceeding.

* Install NixOS

Install NixOS using the flake configuration for =arrakis=:

#+begin_src bash
nixos-install \
  --no-root-passwd \
  --root /mnt \
  --flake github:masood09/nix#arrakis \
  --option tarball-ttl 0
#+end_src

* Cleanup and Reboot

Unmount all filesystems, deactivate LVM, close LUKS, and reboot:

#+begin_src bash
umount -Rl /mnt
vgchange -an vg
cryptsetup close cryptroot
reboot
#+end_src

Remove the USB drive during reboot.

* Post-Install Notes

After reboot:

- You will be prompted for the LUKS passphrase by =systemd-cryptsetup=.
- Verify the disk layout:
  #+begin_src bash
  lsblk
  df -h /nix
  swapon --show
  #+end_src
- Confirm =/= is tmpfs (ephemeral, wiped on each reboot).
- Confirm =/home= persists across reboots (bind-mounted from =/nix/persist/home=).
- Verify hibernate works:
  #+begin_src bash
  systemctl hibernate
  #+end_src
- Connect to WiFi via =nmcli=:
  #+begin_src bash
  nmcli device wifi list
  nmcli device wifi connect "<SSID>" password "<PASSWORD>"
  #+end_src
  Verify:
  #+begin_src bash
  nmcli connection show --active
  #+end_src
- The age public key from the key generation step must be added to =.sops.yaml=
  and secrets re-encrypted before the first deploy via =just deploy=.
